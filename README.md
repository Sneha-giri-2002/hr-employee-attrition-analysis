# HR Employee Attrition Analysis & Retention Dashboard

**End-to-End HR Analytics Project using MySQL and Power BI**

### Objective
Analyzed employee turnover patterns in a dataset of **1,470 employees** to identify key drivers of attrition and provide actionable insights for HR retention strategies.

### Key Insights
- Employees working **OverTime** had **~3x higher attrition**
- Highest attrition in **Sales Department** and **Sales Representative** roles
- **Younger employees (<25)** and **low-income** groups are at high risk
- Job Satisfaction and Work-Life Balance strongly influence retention

### Tools & Technologies
- **MySQL Workbench** – Data cleaning, validation & analysis
- **Power BI** – Interactive dashboard development
- **SQL** – Exploratory Data Analysis

### Project Structure
- `cleaned version.csv` → Cleaned dataset
- `hr attrition.sql` → All SQL queries used for analysis
- `hr attribution analysis.pbix` → Power BI Dashboard
- `screenshots/` → Dashboard images

### How to Run the Project
1. Import `cleaned version.csv` into MySQL
2. Run queries from `hr attrition.sql`
3. Open the `.pbix` file in Power BI Desktop

### Dashboard Features
- Overall KPIs
- Department & Role wise attrition
- Filters by Age, Income, OverTime, etc.
- High-risk employee segment analysis
